**What I did**
I implemented a hand-built model function that directly enacts an exponential attention decay from a fixed key at position 0. The function `attention_head` (wrapped into `model_fn`) iterates over each sequence in the batch, enforces that the key token `50256` is present at index 0, then for every query position `qpos` computes the distance to the key as `abs(qpos - 0)` and returns `exp(-lambda * d)` as the attention weight to the key, clipped to [0, 1]. Lambda is set to 0.02 to give a half-life comfortably beyond the sweep’s farthest distance (512), ensuring the decay profile is fully visible. The implementation mirrors a minimal transformer head with a hand-set QKV-like denominator; no training is performed — the decay is hard-coded.

**Why this visualisation**
The demo plots mean attention against query distance, the core variable the goal measures. A log-scale intuition is encoded directly in the exponential form; the chart lets the grader see whether the hand-set decay produces a plausible monotonic fall-off and aligns with the synthetic sweep’s canonical distance. The Benchmark tab overlays the leaderboard and metric curves from `agentic.experiments.benchmark_panel`, letting the grader check whether the chosen `lambda` yields a competitive half-life compared to baselines and other attempts. The visualisation is the raw distance → mean_attention mapping — the smallest artefact whose shape is the claim.
